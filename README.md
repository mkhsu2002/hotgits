# hotgits v0.5 - GitHub Trend Tracker

一個美觀且現代化的 GitHub 熱門專案追蹤工具，基於 React 19 與 Vite 6 構建，內建 AI 分析功能。

## 🚀 功能特點
- **AI 項目洞察**: 整合 Gemini AI，自動分析熱門專案摘要、亮點與應用。
- **本地收藏夾**: 收藏喜愛的專案，資料同步儲存於本地瀏覽器。
- **多語系支援**: 提供完整的繁體中文與英文切換。
- **實時追蹤**: 向 GitHub Search API 獲取每日、每週、每月的熱門專案。
- **進階元數據**: 顯示開源協議 (License) 與 Issues 數量。
- **極簡美學**: 採用深色模式、毛玻璃效果與平滑動畫。
